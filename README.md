# PizzaShop API

## 💻 Project

This project is a RESTFul API application for a pizzeria written in Bun JS

## 🚀 Technologies

This project was developed with the following technologies:

- Typescript
- Bun
- Elysia
- Drizzle
- PostgreSQL
- Docker

## 🖲️ Getting Started

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```
